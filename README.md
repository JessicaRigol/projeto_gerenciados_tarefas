# Gerenciador de Tarefas — Projeto de Aprendizado em Python
Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Python e Programação Orientada a Objetos (POO).

Durante o desenvolvimento, foram aplicadas técnicas como:
- criação de classes e objetos
- métodos de instância
- encapsulamento de lógica em módulos separados
- leitura e gravação de dados em arquivos JSON
- manipulação de listas de objetos
- criação de um menu interativo no terminal
- implementação de operações CRUD (Create, Read, Update, Delete)
